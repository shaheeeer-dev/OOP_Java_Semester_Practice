✅Problem 1: Animal Hierarchy
Scenario: A zoo wants to manage its animals.
Requirements:
•	Create a base class Animal with name, age, and makeSound() method.
•	Create subclasses Lion, Elephant, and Parrot that override makeSound().
•	Demonstrate polymorphism by calling makeSound() for different animals.
