Composition Practice Problems

Problem 1: Car & Engine

Scenario: A car has an engine.

Requirements:
•	Engine class (engineType, horsepower)
•	Car class creates Engine inside it
•	If Car is destroyed → Engine also destroyed

⸻

Problem 2: House & Room

Scenario: A house contains rooms.

Requirements:
•	Room class (roomNumber, type)
•	House class creates multiple Room objects internally
•	Rooms cannot exist without House

⸻

Problem 3: Computer & CPU

Scenario: A computer has a CPU.

Requirements:
•	CPU class (brand, cores)
•	Computer class creates CPU object
•	CPU cannot exist alone

⸻

Problem 4: School & Classroom

Scenario: A school has classrooms.

Requirements:
•	Classroom class (roomNo, capacity)
•	School class creates Classroom objects
•	Classroom dies with School

⸻

Problem 5: Smartphone & Battery

Scenario: A smartphone has a battery.

Requirements:
•	Battery class (capacity, type)
•	Smartphone class creates Battery internally
•	Battery cannot exist without Smartphone
